# Serenity Summit Guest Guide

Responsive static guest guide converted from the Word/PDF guide for use on a GitHub Pages site.

## Files
- `index.html` - guide content and structure
- `styles.css` - responsive styling and print layout
- `script.js` - table of contents, search, expand/collapse and print controls
- `assets/` - images extracted from the source Word document

## Add to an existing GitHub Pages site
Copy this folder into the repository, for example:

```
/guest-guide/
  index.html
  styles.css
  script.js
  assets/
```

The guide will then be available at:

```
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/guest-guide/
```

Update the framed QR code to point to that URL.

## Notes before publishing
The online guide currently includes the Wi-Fi password because it was present in the supplied guide. If the site is public, anyone with the URL can view it.

The water-heater section reproduces the current source document and should be updated when the planned gas/on-demand system is installed.
